### Clone the project
- Clone with Github Desktop, as it has built-in support for Git LFS.
- Alternately, install [Git LFS](https://git-lfs.com/) and clone with command line:
```bash
git clone https://github.com/Maoyeedy/Qiyu_UnityProject.git
```
